XML Schema Document
•Document element is xs:schema
•Can specify target namespace with targetNamespace attribute
•Missing attribute means no target namespace
<xs:schema xmlns:xs="http://www.w3.org/2001/XMLSchema"
  targetNamespace="http://www.example.com/flights"
  xmlns:fl="http://www.example.com/flights"
  elementFormDefault="qualified">
...
</xs:schema>
