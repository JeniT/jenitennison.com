Instance Document
•Point to schemas from instance document
•xsi:noNamespaceSchemaLocation gives location of schema for elements in no namespace
•xsi:schemaLocation gives location of schemas for elements in a namespace
•Pairs of namespace URIs and locations
•If you don't specify a schema, validators are free to try to find one
•Use built-in schema (e.g. for XML namespace)
•Try resolving the namespace URI
•Schemas are combined for validation
