Restricting Types
•Other types derived by restriction
•Use xs:restriction within xs:simpleType
•Indicate base of derivation through
• base attribute
•Nested anonymous simple type
•Simplest restriction is an enumeration
<xs:simpleType name="UKAirportCities">
  <xs:restriction base="xs:token">
    <xs:enumeration value="London" />
    <xs:enumeration value="Birmingham" />
    ...
  </xs:restriction>
</xs:simpleType>
