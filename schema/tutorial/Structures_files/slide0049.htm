XML Schema: XML Structures
22
Attribute Occurrence
•Attributes declared/referenced within complex types, after content model
•Occurrence indicated by use attribute
•required or optional (default)
•prohibited used when restricting complex types
<xs:element name="depart">
 <xs:complexType>
  <xs:sequence>
   <xs:element ref="airport" />
  </xs:sequence>
  <xs:attribute name="time" type="xs:time"
                use="required" />
 </xs:complexType>
</xs:element>
