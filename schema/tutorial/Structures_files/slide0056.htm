Default and Fixed Values
•Specify default value with default attribute
•Apply if attribute is missing
•Apply if element is present but empty
•Specify fixed value with fixed attribute
•Values must be valid against type
•Can't specify default or fixed value for elements or attributes of xs:ID type
<xs:element name="class" type="flightClass"
            default="economy" />
<xs:attribute name="version" type="xs:decimal"
              fixed="1.0" />
